# Interactive Scatter + Table React App

Instructions inside...

Features
- Scatter plot where each point represents a table row (earthquakes dataset example)
- Controls (dropdowns) to choose numeric variables for X and Y axes
- Scrollable, complete data table with headers
- Row selection: clicking table row highlights the chart point; clicking chart point highlights and scrolls to table row
- Loading indicator while parsing CSV
- State handling demonstrated using Props, React Context, and Zustand (external store)


Setup
1. Create a new folder and paste the files from this canvas into the same folder structure under `src/` and `public/`.
2. Run `npm install` (or `yarn`).
3. Run `npm run dev` and open the Vite URL (usually http://localhost:5173).
